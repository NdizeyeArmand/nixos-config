About

In-house baked configs for Home-Manager and NixOS. Borrowed bits sprinkled on top. Using flakes and flake-parts.

See an overview of the flake outputs by running nix flake show git+https://gitlab.com/Dark_Loon/dotfiles.

Structure

  • home: Home Manager config
  • module: Custom options
  • hosts: Multiple machine configurations
  • secrets: SOPS-encrypted files

Usage

To install NixOS use the nix installer ISO. Then run the following:

git clone gitlab.com/DarkLoon/dotfiles ~/
cd ~/dotfiles

# To build system configuration (alternatively ./bootstrap.sh desktop or ./bootstrap.sh server)
./bootstrap.sh

Use a template:

# Provide a development shell (alternatively nix flake init -t gitlab:Dark_loon/dotfiles#java)
dvt java
dvt java newProject
dvt typst my-project
dvt elm
